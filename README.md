# 📝 Personal Blog Website

A responsive and interactive blog website built using **HTML, CSS, and JavaScript**.  
This project allows users to create, edit, delete, and interact with blog posts.

---

## 🚀 Live Website
👉 https://Mohith2112.github.io/my-blog/

---

## ✨ Features

- 📱 Responsive design (mobile + desktop)
- 🌙 Dark mode toggle
- 📝 Add, edit, and delete blog posts
- ❤️ Like button for posts
- 💬 Comment system
- 🔍 Search functionality
- 📂 Upload images from computer
- 🔐 Simple login system (admin access)
- 💾 Data stored using localStorage
- 🎨 Modern UI with animations

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage
